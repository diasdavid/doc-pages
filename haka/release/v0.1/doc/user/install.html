2. Installation¶

2.1. Download the package¶

First, download a Debian package from the Haka website in the Download section.

See also

If you need a package for another distribution, we will need to build it from the sources. See the section Build for more details about this.

2.2. Install¶

Then to install it, type the following command:

$ sudo dpkg -i haka.deb
